## 1. Общая концепция
**Цель:** превратить классический экономический трактат в интерактивный образовательный опыт, где каждая глава становится мультимедийным модулем с визуализациями, схемами, карточками и интерактивными элементами.

**Язык:** русский  
**Аудитория:** студенты, экономисты, самообразователи, все интересующиеся экономикой  
**Тон:** академический, но доступный; уважающий оригинал, но объясняющий сложное через простое

**Основной принцип работы:**  
Обработка строго по одной главе за итерацию, сохранение единого стиля и компонентов от главы к главе.

## 4. План для Главы I «О разделении труда»

### A. Краткое понимание главы
1. **Фундаментальный принцип** – разделение труда как главный двигатель роста производительности.
2. **Классический пример** – производство булавок: 18 операций vs один неквалифицированный рабочий.
3. **Отраслевое различие** – земледелие менее подвержено разделению, чем промышленность.
4. **Три источника эффективности**:
   - Повышение ловкости от специализации
   - Экономия времени на переключении между задачами
   - Стимул к изобретению машин
5. **Социальный эффект** – взаимозависимость: простая вещь (куртка) – продукт труда тысяч людей.
6. **Философский вывод** – специализация → излишки → обмен → всеобщее благосостояние.

### B. Структура страницы
1. **Заголовочный блок** (название, номер, прогресс)
2. **Ключевые идеи** (6 тезисов в карточках)
3. **Интерактивная схема «Производство булавок»** (18 операций, сравнение производительности)
4. **Сравнительный анализ «Земледелие vs Промышленность»** (таблица, примеры стран)
5. **Три причины роста производительности** (раскрывающиеся блоки с примерами)
6. **Цепочка создания стоимости «Шерстяная куртка»** (интерактивная диаграмма связей профессий)
7. **Цитаты** (ключевые выдержки из главы)
8. **Выводы и навигация** (итоги, переход к следующей главе)

### C. Интерактивные элементы
| Элемент | Описание |
|---------|----------|
| **Схема булавок** | Кликабельные операции → анимация + описание |
| **Сравнительный слайдер** | «До/После» разделения труда |
| **Карточки примеров** | Гвозди, ткач, паровая машина – раскрываются при наведении |
| **Фильтр профессий** | Для цепочки «Шерстяная куртка» (добыча, обработка, транспорт) |
| **Прогресс-освоения** | Пользователь отмечает понятые разделы |
| **Мини-опрос** | 3–4 вопроса для самопроверки с обратной связью |
| **Диаграмма «3 причины»** | Интерактивная круговая диаграмма с деталями по клику |
| **Якорное меню** | Быстрая навигация по разделам при скролле |

### D. Визуальные схемы и изображения
1. **Схема мануфактуры булавок** – векторная иллюстрация в стиле индустриального чертежа XVIII века.
2. **Диаграмма сравнения отраслей** – двойная столбчатая: «Глубина разделения труда».
3. **Инфографика «Три причины»** – три соединённых круга с иконками.
4. **Карта цепочки создания куртки** – сетевая диаграмма профессий и связей.
5. **Иллюстрация «Подросток и паровая машина»** – стилизованная гравюра.
6. **График роста производительности** – две кривые: с разделением труда и без.
7. **Коллаж «Предметы обихода рабочего»** – аннотированные фото элементов.

### E. Готовый веб-контент
(См. полный HTML-код в предыдущем ответе – раздел E.)

**Структура HTML:**
- `<section class="chapter" id="chapter-1">`
- Заголовочный блок с метаданными и прогрессом
- Секции для каждого раздела с `data-section` атрибутами
- Интерактивные элементы с ARIA-ролями
- Адаптивная сетка (Flexbox + CSS Grid)
- Минимальные CSS-анимации (плавные переходы)

### F. Связь с предыдущими/следующими главами
- **← Введение:** Глава I развивает первый из двух условий богатства – «искусство, умение и сообразительность в применении труда».
- **→ Глава II:** Если Глава I описывает **как** работает разделение труда, Глава II отвечает **почему** оно возникает – из склонности человека к обмену.
- **→ Вся книга:** Принцип разделения труда – фундамент для ценообразования, заработной платы, прибыли, капитала.

**Ключ к пониманию всей книги:**  
*Разделение труда – не просто экономический механизм, а базовый социальный феномен, который объясняет:*
1. *Почему общества богатеют (специализация → производительность)*
2. *Как возникает рынок (излишки → обмен)*
3. *Почему люди зависят друг от друга (взаимодополняемость профессий)*

## 6. Технические требования

### Фронтенд
- **HTML5** – семантическая разметка, доступность (ARIA)
- **CSS3** – Grid, Flexbox, CSS Custom Properties (переменные цветов/шрифтов)
- **JavaScript (ES6+)** – для интерактивности, минимально необходимый
- **Адаптивность** – mobile-first подход
- **Производительность** – ленивая загрузка изображений, оптимизация анимаций

### Организация кода
```
/project
  /css
    - design-system.css    # Основные стили, переменные
    - components.css       # Готовые компоненты (карточки, таблицы)
    - chapter-1.css        # Специфичные стили для главы 1
  /js
    - common.js            # Общие функции (навигация, прогресс)
    - chapter-1.js         # Интерактивность для главы 1
  /images
    /chapter-1             # Изображения для главы 1
    /icons                 # Иконки, SVG
  /chapters
    - chapter-1.html       # Полная страница главы 1
    - chapter-2.html       # Глава 2 (будет создана)
  index.html               # Главная страница, оглавление
  PLAN.md                  # Этот документ
```

### Доступность (accessibility)
- Контраст цветов не менее 4.5:1
- Полная навигация с клавиатуры
- ARIA-роли для интерактивных элементов
- Альтернативные описания для изображений
- Логичный порядок фокуса
